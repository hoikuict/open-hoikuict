# 保育園職員が現場から作る、オープンな保育ICT

## 園児名簿、登降園管理、保護者打刻を中心に、導入ロードマップ・セキュリティ最低ライン・園内運用テンプレを公開しながら改善していくプロジェクトです。

保育園職員による保育園職員のための保育ICTを作り上げる取り組みです。

これまでGASで積み上げてきた仕組みが、便利な反面、散らかって引き継ぎも大変になってきました。自戒を込めていったん全部ほどいて、1から組み直します。

オープンにしているのは、現場の意見や協力を得ながら進めていければありがたいからです。
